-   reconstructible: əwɨ
-   parallel innovations:
    -   grammaticalization of emphatic marker \**rə*
    -   coalescence of *wɨ* to *u*
-   appearance of *j* in Pemón and Panare – via /əwɨ/ \> /ɨwɨ/ \> /ɨu/
    \> /ju/? ([Meira 2002](#source-meira2002first): 259)
-   additional *hɨ* in Manapiarean
